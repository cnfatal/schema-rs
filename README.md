# schema-rs

`schema-rs` is a Rust project that dynamically renders native GUIs based on schemas (e.g., JSON Schema). It features a core logic library and an `egui`-based user interface.

## Features

- **Schema-Driven Native GUI**: Dynamically generate and render native UI components directly from schemas.
- **Core Library**: `schema-rs-core` for parsing and processing schema data structures.
- **egui Integration**: `schema-rs-egui` for rendering the schema-driven UI using the `egui` framework.

## Components

- `schema-rs-core`: The core logic and data structures.
- `schema-rs-egui`: The GUI components built with `egui`.
- `examples/playground`: A playground application for testing and demonstration.
